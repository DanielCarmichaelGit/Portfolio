Seat Check

Seat Check was created during the creation of the Boyd Ranch website. Initially ticket tailor was going to be used for processing ticket sales and manaaging event creations. I designed Seat Check to be able to do all of the things ticket tailor can do but better. Seat Check has a 'push to squarespace' button that allows any event created to be instantly pushed as an event right to the desired creator's website. And, it is absolutely free to use.

Fake Transaction Generator

This project was desgned to test my abilities in JavaScript. The process involved calls to an API to get random user data, then the program generated random transaction costs which fully encompassed every detail of a modern transaction including; shipping, shipping costs, units purchased, and more. Each transaction was generated via classes to ensure stable data creation.
